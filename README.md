# CyberStart Academy

A beginner-friendly cybersecurity learning website organized chapter-by-chapter from a video-course flow.

## Chapters

1. Securing Accounts
2. Securing Data
3. Securing Systems
4. Securing Software
5. Preserving Privacy
6. Final Defender Project

## Labs

- Accounts: password and phishing audit
- Data: integrity and backup check
- Systems: login log analysis
- Software: input validation
- Privacy: permission audit

This project is designed for safe, legal learning. All labs use local sample data inside the browser.
