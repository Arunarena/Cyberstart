THE DEFENDER'S CODE

Keep it legal, always.

“Real” cybersecurity practice means real skill, not real victims. Only test systems you own or have written permission to test.

01Get permission first. A public website is not an invitation to test it.

02Use labs and CTFs. Practice on safe platforms designed for learning.

03Report responsibly. If you find a real issue, do not exploit or share private data.
